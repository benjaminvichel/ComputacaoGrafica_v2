| Tecla | Ação                               |
| ----- | ---------------------------------- |
| `ESC` | Fecha o programa                   |
| `X`   | Ativa rotação contínua no eixo X   |
| `Y`   | Ativa rotação contínua no eixo Y   |
| `Z`   | Ativa rotação contínua no eixo Z   |
| `W`   | Move o modelo para frente (Z+)     |
| `S`   | Move o modelo para trás (Z-)       |
| `A`   | Move o modelo para a esquerda (X-) |
| `D`   | Move o modelo para a direita (X+)  |
| `I`   | Move o modelo para cima (Y+)       |
| `J`   | Move o modelo para baixo (Y-)      |
| `O`   | Aumenta a escala do modelo         |
| `L`   | Diminui a escala do modelo         |
