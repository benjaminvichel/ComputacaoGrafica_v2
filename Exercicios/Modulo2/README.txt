Movimentação

W - Move o cubo para frente
S - Move o cubo para tras
A - Move o cubo para a esquerda
D -  Move o cubo para a direita
I - Move o cubo para cima
J - Move o cubo para baixo

Rotação

X - Rotaciona o cubo no eixo X
Y - Rotaciona o cubo no eixo Y
Z - Rotaciona o cubo no eixo Z

Escala

[ - Diminui a escala
] - Aumenta a escala

Para criar novos cubos

N - criar novo cubo no local atual
